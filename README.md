# 🎮 Rock-Paper-Scissors AI Game

A **professional, interactive Rock-Paper-Scissors AI game** built with **Python and Streamlit**.  
Test your luck and strategy against a smart AI that predicts your moves based on your history!

## 🚀 Live Demo

Play the game online: [Open Rock-Paper-Scissors AI](https://rock-paper-sciappr-v9gbo8827agmhqkwmctmz6.streamlit.app/)  


## 🛠 Features

- Interactive UI with **3-column layout** for moves and results  
- **Smart AI** that predicts your next move  
- **Colored results**: Green = Win, Red = Loss, Orange = Tie  
- **Game statistics**: Wins, Losses, Ties displayed in a bar chart  
- **Most Played Moves chart** for tracking your choices  
- **Game history table** showing all rounds with emojis  
- **Reset button** to start fresh anytime  
- Built entirely with **Streamlit, pandas, and matplotlib**
